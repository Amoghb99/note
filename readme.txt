How to use this node js app?

npm i 

To view the full list,
node app.js list

To add a note to the list,
node app.js add --title "titlename" --body "bodyname"

To remove a note from the list,
node app.js remove --title "titlename"


To view a custom note,
node app.js read --title "titlename" 
You will get the item by the title name

All the data is stored in the notes.txt file, so you can have a look at it at any moment.
